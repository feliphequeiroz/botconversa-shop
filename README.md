## 📖 Sobre o Projeto

Esta é a nova versão da landing page do **BotConversa**, reconstruída do zero com foco em clareza visual, velocidade de carregamento e experiência de conversão. O layout anterior (Dark Mode) foi preservado como backup em `index-dark.html`.

O conteúdo foi estruturado com base no arquivo `contexto.txt`, aplicando um **Design System claro e elegante**, com tipografia refinada, sombras sutis e uma paleta alinhada à identidade visual da marca e ao ecossistema WhatsApp.

---

## ✨ Funcionalidades Principais

- 🌓 **Light Mode Premium**: Fundo suave, cards brancos, contrastes otimizados e sombras leves para leitura confortável.
- 💰 **Switch Mensal / Anual**: Toggle interativo que atualiza dinamicamente preços, textos e links de checkout dos planos.
- 📊 **Comparativo "Antes vs Depois"**: Seção visual que contrasta a operação manual com a automação inteligente do BotConversa.
- 🧭 **Guia "Como Escolher?"**: Fluxo prático para ajudar o visitante a decidir entre API Não Oficial e API Oficial da Meta.
- 🖼️ **Assets Otimizados**: Logo em SVG e imagens em AVIF para máxima qualidade com mínimo peso.
- 📱 **100% Responsivo**: Layout fluido e mobile-first, testado em breakpoints modernos.
- ⚡ **Zero Dependências**: HTML, CSS e JS puros. Sem frameworks, sem build step, carregamento instantâneo.

## 🎨 Design System & Tecnologias

| Categoria      | Detalhes                                                                 |
|----------------|--------------------------------------------------------------------------|
| **Fundo**      | `#F8FAFC` (cinza ultra suave) / `#FFFFFF` (cards e superfícies)          |
| **Destaque**   | `#00A884` (WhatsApp Teal) / `#075E54` / `#05cd99` (acentos e CTAs)      |
| **Tipografia** | `Manrope` (UI/Títulos) / `JetBrains Mono` (dados técnicos e preços)      |

| **Bordas**     | `#E5E7EB` (divisores e contornos de cards)                               |
| **Stack**      | HTML5 Semântico • CSS3 (Variables, Grid, Flexbox) • JavaScript Vanilla   |
